# Student Performance Tracker

A command-line Java application to manage student 
records and track academic performance.

## Features
- Add and remove students
- Add multiple grades per student
- Calculate average, highest and lowest grade
- Search student by name
- Save and load data from file (persistent storage)

## Tech Stack
- Java
- OOP (Classes, Encapsulation)
- ArrayList & Collections
- File I/O (BufferedReader / BufferedWriter)

## How to Run
1. Clone the repository
2. Open in VS Code or any Java IDE
3. Run `App.java`
